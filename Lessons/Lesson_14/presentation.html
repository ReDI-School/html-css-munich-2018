CSS Animations

ReDI HTML+CSS

CSS animations consist of two parts:

  1. Keyframes: Define the animation itself
  2. Style property: Define how the animation should be played

Example

See the Pen ReDI HTML: 14-01 Grow by Max (@cpury) on CodePen.

How to make it loop?

  1. Edit the animation: 
    								
@keyframes grow {
	0% {
		font-size: 64px;
	}
	50% {
		font-size: 172px;
	}
	100% {
		font-size: 64px;
	}
 }
  2. Add a setting to the animation property: 
    								
animation: grow 2s infinite alternate;

@keyframes: Syntax

Define an animation step by step:

						
@keyframes animation_name { 
	time_1: { 
		property: value; 
		… 
	} 
	time_2: {
		property: value; 
		… 
	}
	…
 }

animation: Syntax

Define how an animation should be played on an element:


animation: animation_name duration count …;

Define only one animation. Use it with different delays!

You can control how many times an animation should be repeated:


animation: changecolor 1s 1;

  • An animation only sets styles while it's playing
  • Before and after the animation, the element's own styles are applied
  • What if we want to keep that style?

Fill Mode

Possible values:

  • backwards: Apply first animation styles before it is played
  • forwards: Keep final animation styles after it is played
  • both: Keep styles before and after

How to create a bouncing ball?

Let's do it together step by step!

How to make a ball?

							
width: 100px;
height: 100px;
border-radius: 50%;

How to make the ball jump up and down?

  • Try to recreate the movement in a CSS animation
  • Use position: absolute; and top or bottom properties

My solution:

						
@keyframes bounce {
	0% {
		bottom: 60%;
	}
	40% {
		bottom: 5%;
	}
	50% {
		bottom: 0%;
	}
	65% {
		bottom: 5%;
	}
	90% {
		bottom: 55%;
	}
	100% {
		bottom: 60%;
	}
}

How to deform the ball?

transform: scale(1, 2);


transform: scale(2, 1);


transform: scale(3, 3);


transform: scale(0.2, 0.2);


transform: scale(1, 1);



→ scale(width, height)

→ The values are factors!
1 = 100%, 0.5 = 50%, 2 = 200%, etc.
