Which two parts do CSS animations consist of?
→ @keyframes and animation-property

What can go into a keyframe definition?
→ Any CSS style

How to make a div twice as large, no matter what shape it has?
→ transform: scale(2, 2);

How do you set margin-left, -top, -right, -bottom at once?
→ margin: 10px;

How do you change the style when an element is touched by the mouse
→ .my_class:hover { ... }

How do you animate the :hover state?
→ transition: all 1s;

Let's revisit hyperlinks!

What are hyperlinks?
A link to another .html page

How do you add a hyperlink?
<a href="...">...</a>

Example:

							
Click <a href="about-us.html">here</a> to learn more about us!
Click here to learn more about us!

What can go into a href?

  • A complete URL: http://www.google.com
  • Absolute path: /2018/04/08/blogpost.html
  • Relative path: about-us/contact.html
  • Section: #contact

The a-tag uses a lot of pseudo-classes:

a:hover when being hovered

a:active when being clicked

a:visited when it has been visited

By default, a website opens in the current browser window/tab.

You can force the browser to open it in a new tab, like this

<a href="..." target="_blank">...</a>

Directory Structure

A website is like a directory on your computer:
it can have many files and subdirectories.

In a URL, you can access files in a subdirectory with the /

Example:
my-website.com/my-subdirectory/my-file.html

Many HTML pages can share one CSS file.
