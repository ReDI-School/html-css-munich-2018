ReDI HTML+CSS

How do you create a hyperlink?
<a href="...">...</a>

What goes between inside the a-tag?
The HTML that should be clickable, e.g. text or image

What goes into the href attribute?
The URL you want to link to.

How do you link to "page.html" which is in the folder "pages"?
pages/page.html

How do you create a link out of an image?
Put the img-tag inside the a-tag!

What does the CSS selector "a:hover" do?
Style a link while the mouse is hovering it

How do you set the style of a link while it is clicked?
a:active { ... }
