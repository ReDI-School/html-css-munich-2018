How do you create a hyperlink?
<a href="...">...</a>

What goes between inside the a-tag?
The HTML that should be clickable, e.g. text or image

What goes into the href attribute?
The URL you want to link to.

How do you link to "page.html" which is in the folder "pages"?
pages/page.html

How do you create a link out of an image?
Put the img-tag inside the a-tag!

What does the CSS selector "a:hover" do?
Style a link while the mouse is hovering it

How do you set the style of a link while it is clicked?
a:active { ... }

What is 'float'? Think about print design like newspaper or magazine.
In a print layout, text wraps around image as needed. This is called "text wrap".
Magazine design - honor or ignore the image!
Go over or not!
Thats the different between images that belong to the float or not.
Web design is similar
CSS float means if text float around them or not.
Float elements = part of the float of the page
What's the opposite of float?
Absolute
Absolute elements are removed from the float Absolute elements do not affect other elements in the page
Example of a float 
					
						.sidebar{
							float:right
						}
					
				
					
						.sidebar{
							float:left
						}
What are floats used for?
Other than text wrap around image
You can use it to style entire website
Also useful for smaller layout. For example: we can float our avatar image. When the image is resized, text floats around it.
Using relative position on the container and absolute position on the avatar
argh! no text wrap
When do we clear the float?
The sidebar is floated to the right but it is too short. Footer got pushed up!

To fix this problem, the footer can be cleared to ensure it stays beneath both floated columns. 
					
						#footer {
							clear: both;
						}
