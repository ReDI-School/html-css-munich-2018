Which two parts do CSS animations consist of?
→ @keyframes and animation-property

What can go into a keyframe definition?
→ Any CSS style

What happens between keyframes?
→ The browser automatically interpolates

How do you decide the duration of an animation?
→ animation: my_animation 1s;

How can an animation be played infinitely
→ animation: my_animation 1s infinite;

What does this animation do?

    height: 20px;

How to create a bouncing ball?

Let's do it together step by step!

How to make a ball?

							
width: 100px;
height: 100px;
border-radius: 50%;

How to make the ball jump up and down?

  • Try to recreate the movement in a CSS animation
  • Use position: absolute; and top or bottom properties

My solution:

						
@keyframes bounce {
	0% {
		bottom: 60%;
	}
	40% {
		bottom: 5%;
	}
	50% {
		bottom: 0%;
	}
	65% {
		bottom: 5%;
	}
	90% {
		bottom: 55%;
	}
	100% {
		bottom: 60%;
	}
}

Others are more complicated.
For example animation.

Actually, CSS has many properties for animations:


  • animation-name
  • animation-duration
  • animation-delay
  • etc. etc.

animation is just a combination of them all!

Can you translate these propeties to one big animation property?

animation-direction: alternate;
animation-timing-function: linear;
animation-iteration-count: 2;
animation-delay: 1s;
animation-name: dance;
animation-duration: 0.5s;

→ animation: dance 0.5s 1s linear alternate 2;
